GTD Explorer

An interactive dashboard for exploring 50 years of global terrorism data, built to make a large and sensitive dataset explorable.

About the Project

A data visualization project built for a course at FU Berlin, using the Global Terrorism Database — over 200,000 incidents spanning 1970 to 2020. Goal was it to take the dataset and create an interactive dashboard that allows the user to explore the dataset in a more natural way, while trying to include everything people might be interested in.

One of the main challenges was the data quality itself. 1993 is missing entirely, casualty figures are often incomplete, and coverage varies significantly by region aswell as time. Rather than hiding those gaps, the design tries to make them visible and communicate the limitations clearly by adding a big disclaimer before letting people explore the data.

What I built

  • Disclaimer screenbefore the data loads, covering what the dataset includes, what's missing, and its limitations
  • Country Viewto select any country to see incidents on the map alongside breakdowns of attack types, active groups, and trends over time
  • Group Activity Viewto compare up to 5 groups side by side, showing where and when they were active
  • Three map display modesfor clustered markers, a density heatmap, and individual incident dots with hover details
  • combined timelinetracking incidents, deaths, and injuries together

How it came together

The project followed a user-centered design process starting with defining target audiences and working through sketches and layout iterations before writing any code. A central map made the most sense given how geographic the data is, and two main views emerged from thinking about different use cases: one focused on countries, one on specific groups.

After building an initial prototype, I ran a think-aloud study with a small group of users (3 iterations with other students). That session showed some small usability flaws, which I implemented into the final version. Some changes were a darker map style, improved zoom, consistent color use and more. If you'd like to learn more about the project, feel free to check out my GitHub repository or send me an email.
